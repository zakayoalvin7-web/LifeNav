LifeNav is a simple web app to track your daily habits, build streaks, and get motivational tips.
Add habits or tasks
Mark them as done and track streaks
Get AI-powered tips for motivation
View your progress in an easy dashboard
Toggle dark mode for night use
Built with: HTML, CSS, JavaScript, and Chart.js
How to use:
Open index.html in your browser.
Add your habits, mark them as done, and watch your streaks grow!
